Oasis Infobyte – Data Analytics Internship (Level 1)

📌 Project Overview

This project is part of the Oasis Infobyte Data Analytics Internship and focuses on performing Exploratory Data Analysis (EDA) on retail sales data. The objective is to analyze sales patterns, customer behavior, and product performance to uncover meaningful insights that can help businesses make informed, data-driven decisions.

EDA plays a crucial role in understanding data before applying advanced analytics or machine learning models.

🏢 Internship Details

Program: Data Analytics Internship

Organization: Oasis Infobyte

Duration: 4 Weeks

Level: Level 1

Project Type: Hands-on Data Analytics Project

🎯 Project Objectives

Perform data cleaning and preprocessing on retail sales datasets

Generate descriptive statistics to understand sales distribution

Analyze time-based sales trends

Study customer demographics and product performance

Visualize insights using effective charts and plots

Provide actionable business recommendations based on findings

🛠 Tools & Technologies

Python

Pandas & NumPy – Data manipulation

Matplotlib & Seaborn – Data visualization

Jupyter Notebook – Analysis and reporting

📈 Key Insights (Summary)

Certain product categories contribute significantly more to revenue

Sales show noticeable seasonal patterns

A small segment of customers drives a large portion of total sales
